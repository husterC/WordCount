	可执行文件在bin文件夹下，其中input.txt均为输入统计文档，output.txt为输出文档，stopword.txt为停用词文档，compair5.txt为输入统计文档，用于与input5.txt对比。
	jar包在转为exe文件时可能出现错误，若exe文件存在功能无法执行，可在Java环境下重新编译执行。
	指令输入时，如果测试用例文件中指令编码出现问题，可手动输入指令。
	

统计说明：
	在进行字符统计时，将Windows中回车计算为两个字符

	在统计注释行和代码行时，将单个的‘{’或‘}’不算作代码行：若只有‘{’或‘}’，则算作空行，若后面有注释，则算为注释行，若后面有其余有效字符，算为代码行。
即：
{
}
为空行
}//note
为注释行
{main
为代码行

	在统计注释行和代码行时，有一个或以上的除‘{’或‘}’外的有效字符算为代码行
即：
a//note
为代码行
{//note
为注释行
